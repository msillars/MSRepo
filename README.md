# Management System

Personal management system for tracking and controlling workflows across life domains.

## Structure

- **dashboard/** - Web-based dashboard for viewing status across all domains
- **scripts/** - Automation scripts for various tasks
- **data/** - Local data storage (not committed to git)

## Domains

1. **Photography** - RAW files, website, print-on-demand
2. **Work** - Job applications, CVs, projects
3. **Life Admin** - House, finances, utilities
4. **Relationships** - Friends, family connections
5. **Living** - Social, travel, personal enjoyment

## Getting Started

### Dashboard
Open `dashboard/index.html` in your browser to view the management dashboard.

### Scripts
See individual script README files in the `scripts/` directory.

## Setup

1. Clone this repository
2. Install dependencies (per script)
3. Configure paths in relevant scripts
4. Run!

## Infrastructure

- **NAS**: Synology (photography RAWs, backups)
- **Google Drive**: Backup and document storage
- **Local**: Development and automation

Built to persist outside of Claude conversations - sustainable infrastructure for life management.
