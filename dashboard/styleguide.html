Win3x Style Guide

Component library for the Management System.

Use these components consistently across all pages. Copy the HTML structure exactly.

Windows
Active Window
Window Title
Window content goes here.
HTML
<div class="window active"> <div class="title-bar"> <div class="title-bar-controls"> <button class="system-menu" aria-label="System menu"></button> </div> <div class="title-bar-text">Window Title</div> <div class="title-bar-controls"> <button data-minimize aria-label="Minimize"></button> <button data-maximize aria-label="Maximize"></button> </div> </div> <div class="window-body padding"> Window content goes here. </div> </div>
Inactive Window (no .active class)
Inactive Window
This window is inactive (white title bar).
HTML
<div class="window"> <!-- Same structure, no .active class --> </div>
Window (title only - no buttons)
Simple Window
No system menu or min/max buttons.
HTML
<div class="window active"> <div class="title-bar"> <div class="title-bar-text">Simple Window</div> </div> <div class="window-body padding"> Content here. </div> </div>
Buttons
Button States
HTML
<button>Default</button> <button class="primary">Primary</button> <button disabled>Disabled</button>
Button Sizes
Notes
Default min-width: 70px For smaller buttons, override with: style="min-width: auto; padding: 2px 8px;"
Form Inputs
Text Inputs
HTML
<input type="text" placeholder="Text input"> <textarea rows="3"></textarea>
Select / Dropdown
HTML
<select> <option>Option 1</option> <option>Option 2</option> </select>
Checkboxes & Radios
HTML
<div class="checkbox-row"> <input type="checkbox" id="check1"> <label for="check1">Label</label> </div> <div class="radio-row"> <input type="radio" name="group" id="r1"> <label for="r1">Label</label> </div>
Panels & Containers
Raised Panel
Raised panel with 3D effect
HTML
<div class="raised padding"> Content </div>
Lowered Panel (inset)
Lowered/inset panel
HTML
<div class="lowered padding"> Content </div>
Fieldset with Legend
Group Label Content inside fieldset
HTML
<fieldset class="padding"> <legend>Group Label</legend> Content </fieldset>
Layout Utilities
Flex Row with Gap
HTML
<div class="flex-row gap"> <button>Item 1</button> <button>Item 2</button> </div>
Flex with Alignment
HTML
<div class="flex-row gap justify-end"> <button>Cancel</button> <button class="primary">OK</button> </div> Available: .justify-start | .justify-center | .justify-end .align-start | .align-center | .align-end
Spacing Classes
.padding (8px all sides)
.margin (8px all sides)
Available Classes
.padding 8px all sides .padding-top 8px top .padding-right 8px right .padding-bottom 8px bottom .padding-left 8px left .padding-horizontal left + right .padding-vertical top + bottom .margin (same pattern as padding)
Modal Dialog Pattern
Modal (shown inline for demo)
Dialog Title

Dialog content here.

Double-click system menu to close.

HTML
<div class="modal"> <div class="window active"> <div class="title-bar"> <div class="title-bar-controls"> <button class="system-menu" ondblclick="close()"></button> </div> <div class="title-bar-text">Title</div> <div class="title-bar-controls"> <button data-minimize></button> <button data-maximize></button> </div> </div> <div class="window-body padding"> Content <div class="flex-row gap justify-end"> <button>Cancel</button> <button class="primary">OK</button> </div> </div> </div> </div>
