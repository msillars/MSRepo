Critical Constraint: Square Buttons

Buttons must always remain square (20x20px). This is enforced in CSS with explicit width AND height constraints.

In Windows 3.1, the title bar height (SM_CYCAPTION) was a system constant. Buttons were 16x16px squares that fit within this fixed height. Our implementation scales this to 20x20px for modern displays.
