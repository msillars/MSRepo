Active/Inactive States

State Title Bar Frame
Active Blue (#0000C0) background, white text Gray (#C0C0C0) - unchanged
Inactive White (#FFFFFF) background, black text Gray (#C0C0C0) - unchanged

Key Point

The window frame (gray border area) does not change when the window is active or inactive. Only the title bar changes color.

Interactive Behavior

Action Trigger Result
Move Drag title bar Window repositions
Resize Drag window edges/corners Window changes size
Minimize Click minimize button Window collapses to icon
Maximize Click maximize button Window fills container
Restore Click maximize on maximized window Returns to previous size
Close Double-click system menu Window closes
Focus Click anywhere on window Brings to front, becomes active

See: window-demo.html for interactive behavior demo.

CSS Classes

Class Purpose
.window Base window container
.window.active Active (focused) window state
.window.draggable Enables interactive positioning
.window.minimized Hidden state (minimized to icon)
.window.maximized Full container size
.dialog Dialog window variant
.title-bar Title bar container (22px)
.window-body Main content area
.status-bar Bottom status bar (20px)
